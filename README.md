# Reminder Website – PHP MVC Application

Reminder Website is a PHP-based web application built using the Model-View-Controller (MVC) architecture.  
It allows users to register, log in securely, and manage their personal reminders.

The project demonstrates authentication, session handling, database integration, and structured MVC design.

## Architecture

The project follows a custom MVC structure:

```
reminderwebsite/
│
├── app/
│   ├── controllers/
│   ├── models/
│   ├── views/
│   └── core/
│
├── public/
│   ├── index.php
│   └── .htaccess
│
└── README.md
```

## Tech Stack

- PHP (Core PHP)
- MVC Architecture
- MySQL
- HTML5 / CSS3
- Bootstrap
- Session-based Authentication

## Features

- User Registration
- Secure Password Hashing
- Login with Validation
- Account Lockout After Multiple Failed Attempts
- Session Management
- Create, View, Edit, Delete Reminders
- Admin Reporting
- Clean MVC Routing System

## Security Features

- Passwords hashed using PHP `password_hash()`
- Login attempt tracking
- Temporary account lock after multiple failed attempts
- Server-side validation
- Session protection

## How to Run on Replit

This project runs directly on Replit — no XAMPP required.

### Steps:

1. Copy the GitHub Clone URL
2. Go to Replit
3. Click Import Code
4. Select Import from GitHub
5. Paste the Clone URL
6. Click Import

The application will start automatically in the browser preview.
